# Job Application Portal

A web-based recruitment system built with PHP and MySQL.

## Features

* User Registration
* Secure Login & Logout
* Password Hashing
* Job Listings
* Job Applications
* CV Uploads
* Admin Dashboard
* Application Approval/Rejection

## Technologies

* PHP
* MySQL
* Bootstrap 5
* HTML/CSS

## Installation

1. Install XAMPP
2. Create a MySQL database named `job_portal`
3. Import the SQL tables
4. Update `db.php`
5. Run via `http://localhost/job-portal`
